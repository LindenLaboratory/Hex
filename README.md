# Hex
Hex is an AI-powered search engine made for efficiently researching topics. It scrapes the web and summerises results. Hex is made to be run with a simple frontend (see [sites.html](sites.html) for a simple html program you can use in google sites) that sends the ready-made backend the search query. There is also a database also powered by python stored in a .csv files (Hex gathers search information about logged in users and sends it back to the database.
